# bli

a lightweight package manager for isolated project environments with snapshotable state and a minimal cli built around cargo-style semantics

---

**features:**

- **cargo-based semantics:** the command wording / styling makes you feel like you're loading packages onto a truck
- **isolated environments:** create small project environments for development, deployment, etc.
- **package snapshots:** sleek snapshots save sorry system states!
- **add other sources:** some savor supplementary sources so sourly, so we stashed them in bli :D

---

bli is still in very early development, so there's no documentation, databases, and the project may seem mostly dormant.
